Lists and Tables

HTML for Lists

  • Ordered list: <ol>
  • Unordered list: <ul>
  • Discription list: <dl>
  • list item: <li>
  • description term: <dt>
  • description data: <dd>

CSS for Lists

- table: <table> - table row: <tr> - table header: <thead> - table header cell: <th> - table body: <tbody> - table footer: <tfoot> - table footer cell: <td>
